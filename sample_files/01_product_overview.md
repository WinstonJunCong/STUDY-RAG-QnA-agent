# NovaDesk — Product Overview

NovaDesk is an all-in-one customer support platform designed for small to mid-sized businesses. It combines ticketing, live chat, a self-service knowledge base, and analytics into a single, unified workspace.

## Core Modules

- **Ticketing System** — Manage customer requests from email, web forms, and social media in one inbox.
- **Live Chat** — Embed a chat widget on your website or mobile app for real-time support.
- **Knowledge Base** — Publish help articles, FAQs, and how-to guides that customers can search 24/7.
- **Analytics Dashboard** — Track ticket volume, response times, customer satisfaction (CSAT) scores, and agent performance.
- **Automation Engine** — Set up rules to auto-assign tickets, send canned responses, and escalate priority issues.

## Who Is NovaDesk For?

NovaDesk is built for:
- SaaS companies with growing support queues
- E-commerce stores handling order and return inquiries
- Professional services teams managing client requests
