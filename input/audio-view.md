# Audio view

The audio view displays a [timeline](https://en.esotericsoftware.com/spine-keys#Timeline) and the waveform for [audio events](https://en.esotericsoftware.com/spine-events#Audio-events) keyed in the animation.

![Screenshot: View](https://en.esotericsoftware.com/img/spine-user-guide/audio-view/view.png)

See the audio [example projects](https://en.esotericsoftware.com/spine-examples#Audio) for usage examples.

# Audio events

Each audio event key in the active animations is shown in a list with a unique color.

![Screenshot: Audio List](https://en.esotericsoftware.com/img/spine-user-guide/audio-view/audio-list.png)

Selecting an audio event key in the list will set the timeline position to that key and dim the other waveforms.

If the corresponding [audio event](https://en.esotericsoftware.com/spine-events#Audio-events) in the tree has its visibility dot cleared then it won't appear in the audio view and the audio won't be played.

# Volume

The volume slider controls the volume in the Spine editor for all audio events. The volume can be also muted.

![Screenshot: Volume](https://en.esotericsoftware.com/img/spine-user-guide/audio-view/volume.png)

To set the volume for a specific audio event, see [audio event volume](https://en.esotericsoftware.com/spine-events#Volume).

# Audio device

This select box controls which audio device the Spine editor will use to play audio.

![Screenshot: Audio Device](https://en.esotericsoftware.com/img/spine-user-guide/audio-view/audio-device.png)
